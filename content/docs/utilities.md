### To center, left or right align text use these classes:

To center it

```css
.center-text
```

To align left

```css
.left-text
```

To align right

```css
.right-text
```

### To hide elements

To hide on tablet and down

```css
.hide-tablet-down
```

To hide on mobile only

```css
.hide-mobile
```

To hide on any screen size

```css
.hide
```
